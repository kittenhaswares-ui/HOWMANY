# Privacy

HOWMANY does not send anything over the network and has no backend or community database.

The plugin reads only the local game state needed to render its PvP overlay: whether the player is in PvP, visible player objects, locally available hard/cast targets, harmful action outcomes affecting the local character, and job IDs. These observations are held only for a few seconds and are never saved. Dalamud stores only the user's local display settings.
